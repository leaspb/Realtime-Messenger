# Цель и границы проекта

## Цель

Создать простое приложение для мгновенного входа в комнату и общения в реальном времени:
- текстовый чат внутри комнаты;
- голосовой звонок между участниками через peer-to-peer WebRTC;
- минимальный порог входа (без регистрации и сложной авторизации).

## Целевая аудитория

- небольшие команды и группы;
- demo/PoC сценарии для real-time коммуникаций;
- учебный проект по WebSocket/WebRTC.

## Границы текущей версии

### Входит в scope
- вход в комнату по `roomId` и `username`;
- список активных пользователей в комнате;
- сообщения в реальном времени;
- управление голосом: старт звонка, mute/unmute, завершение звонка;
- health endpoint `GET /api/health`.

### Не входит в scope
- регистрация/логин пользователей и роли;
- хранение истории чатов в production-ready виде;
- модерация, файлы, видео, screen sharing;
- гарантированная доставка сообщений и офлайн-очереди;
- горизонтальное масштабирование сигналинга.

## Критерии успеха

- пользователь открывает приложение и заходит в комнату за 1-2 действия;
- сообщения видны всем участникам комнаты с минимальной задержкой;
- участники могут установить голосовое соединение без ручной настройки;
- сервер отвечает на health-check и стабильно обслуживает WebSocket подключения.
