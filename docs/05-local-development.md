# Локальная разработка

## 1. Установка зависимостей

```bash
npm install
```

## 2. Подготовка переменных окружения

Минимально необходимо задать `DATABASE_URL`, иначе приложение завершится с ошибкой на старте.

Пример:

```bash
export DATABASE_URL="postgres://user:password@localhost:5432/realtime_messenger"
export PORT="5000"
```

## 3. Запуск в режиме разработки

```bash
npm run dev
```

Что происходит:
- backend стартует через `tsx server/index.ts`;
- Vite подключается как middleware;
- UI доступен на `http://localhost:5000`.

## 4. Проверка работоспособности

1. Открыть `http://localhost:5000` в двух вкладках.
2. Войти в одну и ту же комнату под разными именами.
3. Проверить:
- отображение участников;
- отправку/получение текстовых сообщений;
- запуск/остановку голосовой связи.

## 5. Проверка типов

```bash
npm run check
```

## 6. Работа с Drizzle

```bash
npm run db:push
```

Команда публикует схему из `shared/schema.ts` в БД по `DATABASE_URL`.
