# Быстрое Развертывание - 5 Минут до Продакшена

Самые быстрые способы развернуть Realtime Messenger в интернете.

## Вариант 1: Railway (Проще Всего - 2 минуты) ⭐

**Стоимость:** $0 (первый месяц с $5 кредитами)

1. Загрузите код на GitHub
2. Перейдите на [railway.app](https://railway.app)
3. Нажмите "New Project" → "Deploy from GitHub repo"
4. Выберите ваш репозиторий
5. Готово! Railway автоматически деплоит

**Переменные окружения:** Автоматически устанавливаются Railway

**Ваше приложение:** `https://your-app.up.railway.app`

---

## Вариант 2: Render (100% Бесплатно - 3 минуты)

**Стоимость:** $0 навсегда (с холодными стартами)

1. Перейдите на [render.com](https://render.com)
2. Нажмите "New +" → "Web Service"
3. Подключите GitHub репозиторий
4. Настройте:
   - **Build:** `npm install && npm run build`
   - **Start:** `npm start`
   - **Plan:** Free
5. Разверните!

**Ваше приложение:** `https://realtime-messenger.onrender.com`

**Примечание:** Бесплатный тариф засыпает после 15 мин неактивности (время пробуждения 30с)

---

## Вариант 3: Fly.io (Бесплатный Tier - 5 минут)

**Стоимость:** $0 (3 бесплатных VM)

```bash
# Установить Fly CLI
brew install flyctl  # macOS
# или
curl -L https://fly.io/install.sh | sh  # Linux

# Войти
fly auth login

# Развернуть
cd /path/to/Realtime-Messenger
fly launch
# Следуйте подсказкам, выберите регион, разверните

# Ваше приложение запущено!
fly open
```

**Ваше приложение:** `https://realtime-messenger.fly.dev`

## После Развертывания

1. **Протестируйте WebSocket:**
   - Откройте ваше приложение
   - Введите имя комнаты и имя пользователя
   - Попробуйте голосовой чат

2. **Поделитесь ссылкой:**
   - Отправьте URL друзьям
   - Они могут присоединиться к вашей комнате мгновенно

3. **Мониторинг:**
   - Railway: Встроенная панель управления
   - Render: Логи в панели управления
   - Fly.io: `fly logs`
